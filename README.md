# macFX [![language: English](https://img.shields.io/badge/-English-blue?style=social&logo=markdown)](README_en.md)

使用JavaFX模拟macOS。

## 环境
* 仅在Java8上做过测试，高版本无法保证运行效果。

## 运行

```
mvn compile exec:java
```
